# terryso/homebrew-tap

Homebrew Tap for [ProxyCast](https://github.com/aiclientproxy/proxycast).

## 安装

```bash
brew tap terryso/homebrew-tap
brew install proxycast
```

## 常用命令

```bash
# 更新
brew upgrade proxycast

# 卸载
brew uninstall proxycast

# 移除 tap
brew untap terryso/homebrew-tap
```

## 关于 ProxyCast

ProxyCast 是一个 AI API 代理桌面应用，支持将 AI 客户端凭证转换为 OpenAI 兼容接口。

- **项目地址**: https://github.com/aiclientproxy/proxycast
- **默认端口**: 8999
- **配置目录**: `~/.config/com.proxycast.app/`
